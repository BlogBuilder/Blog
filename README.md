开发工具
- JQuery 2.1.0+
- Bootstrap 3.3.7+


### 安装Node与包管理工具（基于Brew）
```shell
brew install node
```

Step 1. 创建一个项目文件夹 （将Blog Git Clone至本地）
```shell
$ mkdir blog
$ cd blog
$ git clone git@github.com:qulongjun/Blog.git .
```

Step 2. 开始构建
```shell
$ npm install
$ npm run dev
```

Step 3. 访问地址
```shell
$ http://localhost:8888
```
